## CAM_类中心_加权实验

### 一.  数据距离度量

先利用了**欧式距离**进行实验。

1.Single-linkage:要比较的距离为元素对之间的最小距离

```python
def single(a,b):
    min = 1000000;
    for i in a:
        for j in b:
            s = np.sqrt(np.sum(np.square(i-j)))
            if(min>s):
                min=s
    return min  
```
